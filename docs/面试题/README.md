# 面试题

这里是面试题总入口页，适合快速定位专题、按路径复习和临近面试前冲刺。

## 如何使用

- 从左侧完整导航进入具体题目页。
- 先按下方推荐路径过一遍体系，再按岗位方向做深挖。
- 每个专题内优先掌握「原理 -> 场景 -> 追问 -> 取舍」。

## 推荐复习路径

1. [计算机基础面试题](/面试题/计算机基础面试题/)：打牢网络、操作系统、数据结构基础。
2. [Java常见面试题](/面试题/Java常见面试题/)：巩固语言、集合、并发、JVM。
3. [Java框架常见面试题](/面试题/Java框架常见面试题/)：聚焦 Spring 生态和工程实践。
4. [数据库常见面试题](/面试题/数据库常见面试题/)：补齐 MySQL 与 Redis。
5. [微服务和分布式面试题](/面试题/微服务和分布式面试题/)：强化分布式治理与中间件。
6. [项目问题](/面试题/项目问题/)：沉淀可讲清楚的项目叙事与优化案例。

## 面试前自检

- 能否在 3 分钟内讲清一个系统的核心链路。
- 能否说清一个技术选型的收益、代价和边界。
- 能否给出一个真实性能/稳定性优化案例与量化结果。
- 能否回答同一问题的基础版、进阶版、追问版。
