# 算法学习路线

面向工程实践的算法与数据结构路线，建议长期坚持。

## 学习目标

- 掌握常见数据结构与算法思想，具备独立解题能力

## 知识模块

1. 算法基础
   - 时间/空间复杂度、递归与迭代、分治与贪心
2. 数据结构
   - 栈队列、链表、二叉树/搜索树、堆、哈希、图
3. 常见题型
   - 双指针、滑动窗口、二分、回溯、动态规划

## 刷题建议

- 每日 2 题：一道基础 + 一道进阶（LeetCode/Codeforces）
- 记录题解与总结，形成个人知识库

## 下一步

- 配合 [系统设计](/学习路线/系统设计.md) 理解工程中的算法应用
