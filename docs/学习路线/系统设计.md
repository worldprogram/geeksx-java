# 系统设计学习路线

面向业务与工程实践的系统设计能力建设。

## 学习目标

- 理解高并发/高可用设计，掌握缓存、消息队列与分布式基础

## 知识模块

1. 架构基础
   - 分层/分模块、可观测性、容量规划与弹性
2. 高并发与高可用
   - 限流/熔断/降级、冗余与故障转移、灰度发布
3. 缓存与消息队列
   - 本地/分布式缓存、缓存一致性
   - MQ 模型、刷库/重试/去重与顺序性
4. 微服务与治理
   - 服务注册发现、配置中心、网关与安全

## 实战建议

- 设计并实现一个简化的订单系统，涵盖缓存、消息队列与幂等
