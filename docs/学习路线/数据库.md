# 数据库学习路线

建议 2–4 周掌握关系型数据库（以 MySQL 为主）。

## 学习目标

- 能写出正确高效的 SQL 并理解事务与锁
- 会使用索引与执行计划进行性能优化

## 知识模块

1. SQL 基础
   - DDL/DML、查询与聚合、子查询与连接
2. 事务与锁
   - 事务隔离级别、MVCC、行锁/表锁、死锁
3. 索引与优化
   - B+Tree、覆盖索引、联合索引、回表与排序
   - Explain 执行计划、慢查询分析
4. 备份与容灾
   - 主从、读写分离、数据一致性

## 实战建议

- 为博客系统设计表结构与索引策略，并对热点查询做优化
