# 学习路线总览

本章节提供从零到一的系统学习路线，覆盖 Java 入门与进阶、数据库、常用框架、计算机基础、算法与数据结构、系统设计，以及面试冲刺复盘，帮助你高效构建完整的技术能力图谱。

## 路线图

- Java 入门：语法基础、面向对象、常用 API、集合、IO
- Java 进阶：并发编程、JVM、网络编程、泛型与反射
- 数据库：MySQL 基础与优化、事务与锁、索引与执行计划
- 框架：Spring/Spring Boot、ORM（MyBatis/JPA）、Web/REST 设计
- 计算机基础：操作系统、计算机网络、数据结构
- 算法：基础算法、刷题方法论、复杂度与优化
- 系统设计：高并发、高可用、缓存与消息队列、微服务
- 面试冲刺：高频题目、项目亮点包装、答题策略
