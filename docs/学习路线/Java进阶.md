# Java 进阶学习路线

面向有基础的同学，建议 3–6 周完成。

## 学习目标

- 能编写并发安全的程序，理解 JVM 原理
- 掌握网络编程与常用框架的核心机制

## 知识模块

1. 并发编程
   - 线程模型、锁与并发容器、原子与 CAS
   - 基本并发设计模式与问题排查
   - 参考：Java 并发编程（线程、锁、并发容器）
2. JVM 与性能
   - 内存结构、类加载、垃圾回收、JIT
   - 常见性能问题定位思路
   - 参考：JVM 原理（内存、类加载、GC、JIT）
3. 网络编程
   - TCP/UDP、BIO/NIO、常见网络协议
4. 语言高级特性
   - 泛型、反射、注解、函数式编程

## 实战建议

- 实现一个并发任务调度器（线程池、重试、超时）
- 编写压测脚本，观察 GC 日志与吞吐/延迟变化
