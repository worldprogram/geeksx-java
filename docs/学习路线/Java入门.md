# Java 入门学习路线

面向初学者的 Java 基础学习路线，建议 2–4 周完成。

## 学习目标

- 掌握 Java 语法与基础类型
- 理解面向对象三大特性与常用 API
- 熟悉集合与 IO 基础，能进行简单控制台与文件程序开发

## 知识模块

1. 语法与基础
   - 变量与数据类型、流程控制
   - 常用 API（String、Math、时间）
   - 参考：[Java基础](/Java/Java知识学习/Java基础.md)
2. 面向对象
   - 封装、继承、多态，接口与抽象类
   - 参考：[面向对象](/Java/Java知识学习/Java基础.md)
3. 集合框架
   - List/Set/Map 基本使用与底层结构概览
   - 参考：[Java集合](/Java/Java知识学习/Java集合.md)
4. IO 基础
   - 字节流/字符流、常见文件读写
   - 参考：IO 基础（文件读写、流的分类）

## 练习建议

- 完成 3 个小项目：通讯录管理、文件去重、日志分析
- 每天 1–2 道基础算法题（数组/字符串）
