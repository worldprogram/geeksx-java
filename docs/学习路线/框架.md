# 框架学习路线

以 Spring / Spring Boot 为核心，建议 3–5 周。

## 学习目标

- 掌握依赖注入、AOP、事务管理等核心机制
- 能使用 ORM 构建常规 Web/REST 服务

## 知识模块

1. Spring 核心
   - Bean 生命周期、IOC/DI、AOP 原理
   - 事务传播与回滚、声明式与编程式事务
2. Spring Boot
   - 自动装配原理、配置管理、Starter 机制
3. ORM
   - MyBatis 与 JPA 的差异与使用场景
   - Mapper/Repository 设计与分页、查询优化
4. Web/REST
   - 控制器、过滤器/拦截器、中间件设计
   - 统一异常处理与返回结构

## 实战建议

- 构建一个简化版博客/商城后台，包含鉴权、CRUD、分页与日志
