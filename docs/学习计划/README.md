# 📅 学习计划

这里为你制定了不同阶段、不同目标的详细学习执行计划。

## 🚀 阶段计划

- [Java 基础速成（30天）](/学习计划/P1-Java基础速成.md)
  - 适合：零基础或转行同学
  - 目标：掌握 Java 核心语法与常用 API，完成简单项目
- [Java 进阶修炼（60天）](/学习计划/P2-Java进阶修炼.md)
  - 适合：有一定基础，想深入底层的同学
  - 目标：掌握并发、JVM、Netty 及设计模式

## 🎯 突击计划

- [面试突击冲刺（14天）](/学习计划/P3-面试突击冲刺.md)
  - 适合：即将面试的同学
  - 目标：高频八股文背诵、算法题突击、项目亮点梳理
- [架构师成长（长期）](/学习计划/P4-架构师成长.md)
  - 适合：工作 3 年+，寻求技术突破
  - 目标：分布式架构、中间件原理、系统设计
