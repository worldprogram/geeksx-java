# ⚡ 面试突击冲刺计划（14天）

适合面试前 2 周进行高强度复习，聚焦高频考点。

## Day 1-3: Java 基础与集合（夯实地基）

- **复习重点**:
  - `equals` vs `==`，`hashCode` 契约
  - String 常量池，Integer 缓存
  - List/Set/Map 选型与源码（HashMap 必问）
  - Fail-Fast 机制与遍历
- **刷题**: 链表反转、环形链表、LRU 缓存

## Day 4-6: 并发与 JVM（重难点攻克）

- **复习重点**:
  - 线程池参数与配置经验
  - `synchronized` 锁升级过程，`volatile` 可见性
  - JVM 内存区域，GC Roots，垃圾回收算法
  - 类加载过程，双亲委派破坏场景
- **刷题**: 生产者消费者模型、交替打印、死锁模拟

## Day 7-9: 数据库与 Redis（后端核心）

- **复习重点**:
  - MySQL 索引（B+树）、事务隔离级别、MVCC、锁（行锁/表锁/间隙锁）
  - SQL 优化（Explain 使用、索引失效场景）
  - Redis 数据结构与场景（String/Hash/List/Set/ZSet）
  - Redis 持久化（RDB/AOF）、缓存穿透/击穿/雪崩、分布式锁
- **刷题**: 二叉树遍历、层序遍历、第 K 大元素

## Day 10-12: 框架与系统设计（综合能力）

- **复习重点**:
  - Spring Bean 生命周期、循环依赖解决、AOP 原理
  - SpringBoot 自动装配原理
  - 消息队列（Kafka/RocketMQ）积压、丢失、重复消费
  - 分布式 ID、CAP/BASE、分布式事务（TCC/Seata）
- **场景题**: 秒杀系统设计、短链接生成、排行榜设计

## Day 13: 项目梳理与简历优化

- **动作**:
  - 使用 STAR 法则重写项目经历
  - 准备 3 个项目难点与解决方案（自圆其说）
  - 模拟自我介绍（1分钟/3分钟版本）

## Day 14: 模拟面试与心态调整

- **动作**:
  - 找朋友或对着镜子进行模拟面试
  - 查漏补缺，浏览错题本
  - 调整作息，准备好面试环境/着装

## 💡 备战技巧

1. **不求甚解**：时间紧迫，对于极度生僻的底层细节，只需记住结论。
2. **输出倒逼输入**：尝试用自己的话讲清楚一个知识点。
3. **自信**：面试是平等的交流，展现出你的思考过程比结果更重要。
