# 🚀 Java 基础速成计划（30天）

本计划旨在帮助零基础同学快速掌握 Java 核心，具备开发简单应用的能力。

## 第 1 周：环境与语法基础

- **Day 1-2**: Java 历史、JDK 安装、Hello World、IDEA 安装与使用
- **Day 3-4**: 变量、数据类型、运算符、流程控制（if/switch/for/while）
- **Day 5**: 数组的定义与常见操作（遍历、排序）
- **Day 6**: 方法的定义、调用、重载、递归初步
- **Day 7**: **周总结与小测验**（控制台计算器、猜数字游戏）

## 第 2 周：面向对象核心

- **Day 8-9**: 类与对象、封装、构造方法、this 关键字
- **Day 10-11**: 继承（extends）、super、方法重写、抽象类（abstract）
- **Day 12-13**: 多态、接口（interface）、内部类基础
- **Day 14**: **周总结**（设计一个简单的“动物园”或“图书管理”类结构）

## 第 3 周：常用 API 与 集合

- **Day 15**: Object 类常用方法、String/StringBuilder 源码与使用
- **Day 16**: 常用工具类（Math、System、Arrays）、包装类
- **Day 17**: 时间日期类（Date、Calendar、JDK8 新时间 API）
- **Day 18**: 异常处理机制（try-catch-finally、throw/throws）
- **Day 19-20**: 集合框架（Collection、List、ArrayList、LinkedList）
- **Day 21**: Set（HashSet）、Map（HashMap）基础使用与遍历

## 第 4 周：IO、多线程与实战

- **Day 22-23**: IO 流基础（File、字节流、字符流、缓冲流）
- **Day 24-25**: 多线程基础（Thread、Runnable、生命周期、同步 synchronized）
- **Day 26**: 网络编程基础（Socket 通信原理、TCP 简单案例）
- **Day 27-29**: **综合实战项目**（如：控制台版《学生管理系统》或《多用户即时聊天室》）
- **Day 30**: 总结回顾，查漏补缺，准备进入进阶阶段

## ✅ 验收标准

1. 能脱离 IDE 手写常见算法（冒泡排序、二分查找）
2. 理解面向对象思想，能画出简单的类图
3. 熟练使用 List 和 Map 解决数据存储问题
4. 完成至少一个 500 行代码以上的综合项目
